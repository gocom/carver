h1. Carver

Textpattern CMS command line interface. *Work in progress*.

h2. Developing

After cloning install the dependencies with Composer:

bc. $ composer install

Run the CLI by running the @carver@ bin:

bc. $ ./bin/carver help

h2. The plan

bc. $ carver txp:events
$ carver txp:trigger event
$ carver txp:set_pref name value
$ carver plugin:compile
$ carver plugin:init
