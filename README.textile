h1. Carver

"Packagist":https://packagist.org/packages/textpattern/carver | "Issues":https://github.com/gocom/carver/issues

Textpattern CMS command line interface. *Work in progress*.

h2. Install

Using "Composer":http://getcomposer.org:

bc. $ composer require textpattern/carver

h2. Usage

bc. $ ./vendor/bin/carver help

h2. The plan

bc. $ carver txp:events
$ carver txp:trigger event
$ carver txp:set_pref name value
$ carver plugin:compile
$ carver plugin:init
