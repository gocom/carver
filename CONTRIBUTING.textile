"Semantic versioning":http://semver.org/.

h2. Setting up dependencies

After cloning install the dependencies with Composer:

bc. $ composer install

h2. Running

Run the CLI by running the @carver@ bin:

bc. $ ./bin/carver help

h2. Compiling a PHAR

Using "Box":https://github.com/kherge/php-box:

bc. $ box build
